Siphon: CoffeeScript Programming Environment

Siphon is a CoffeeScript Programming Environment, especially for iPad-like Tablet.

Enjoy Siphon by clicking here!

Features

  • Real-time compiling same as the CoffeeScript site
  • REPL (Read-Eval-Print-Loop) in the editor
  • Software keyboard enhancement with flick gesture
    • Click sound is available online.
  • jQuery Mobile look and feel
  • Editor powered by CodeMirror
    • Syntax highlight
    • Undo/redo (⌘-z/⌘-y)
    • Cut/copy/paste (⌘-x/⌘-c/⌘-v)
    • (optional) Smart indent
    • Smart home button: moving to first non-white-space character.
    • Match brackets
  • "File" storage
  • Offline application
  • Available libraries which you can use for programming
    • jQuery library
    • jQuery Mobile library
  • Available libraries which you can import for programming
    • Functional library - Underscore.js (1.2.3 / MIT)
    • Visualization library for canvas - Processing.js APIs (1.3.6 / MIT): You can use the canvas named "canvas" as html id in the execution page.
    • Chart library for canvas - AwesomeChartJS (11/28/2011 / Apache 2.0)
    • Visualization library for SVG - Raphaël (2.0.1 / MIT)
    • Chart library for SVG - elycharts (2.1.4 / MIT)
  • Tutorials by other authors
      • Cheat sheet based on The Little Book on CoffeeScript by A. MacCaw.
      • Smooth CoffeeScript by E. Hoigaard
      • Underscore.js by J. Ashkenas

    Tips

    • Inputting single or double quotation: You can input single or double quotation by flicking comma(') or period(.) key on software keyboard respectively.
    • Missing keys: Several buttons, such as '(', '=', and etc., are also flickable. Touch and hold, then you will see a different key.

    Action Items

    • "Require" support
    • Prelude library in "Smooth CoffeeScript"
      • WebSockets library - ws.coffee (MIT)
      • QuickCheck library - qc.js (2009 / MIT)
    • Code completion
